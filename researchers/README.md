# researchers

if you'd like access to relay data without running a proposer or builder, use:

```
# mainnet
https://relay-analytics.ultrasound.money
# hoodi
https://relay-analytics-hoodi.ultrasound.money
```

standard routes are documented at <https://flashbots.github.io/relay-specs/#/Data>.

we're happy to provide more on request — some undocumented APIs cover the common needs of other relays and builders.

servers are in the EU. no rate limits are enforced but please try not to exceed ~100Mbit/s.

as the relay operates in multiple geographies, it can take seconds to minutes for data to appear in the aggregated view served at `/relay/v1/data` and `/ultrasound/v1/data`.

questions: [@ultrasoundrelay on telegram](https://t.me/ultrasoundrelay) or [contact@ultrasound.money](mailto:contact@ultrasound.money).
